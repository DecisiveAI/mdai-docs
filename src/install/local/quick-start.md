## Getting Started

To begin visualizing your telemetry pipelines using an MDAI Cluster, you will need to complete the following steps:

1. 🛠️ Configure it
2. 🚀 Deploy it (locally)
3. 📈 Send telemetry to it
4. 🔑 Access it
5. ✅ Verify it!
